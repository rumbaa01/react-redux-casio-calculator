# Calculator Casio SL-300SV

## Steps to start:

1. Clone repository
```bash
git clone https://github.com/rumbaa01/react-redux-casio-calculator.git
```

2. Install dependencies

```bash
cd react-redux-casio-calculator && npm install
```

3. Run

```bash
npm start
```

## Used dependencies:

* react
* redux
* create-react-app
